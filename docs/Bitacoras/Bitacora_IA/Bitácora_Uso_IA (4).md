## 2️⃣ Objetivo del uso de IA

La inteligencia artificial se utilizó como apoyo en diferentes tareas relacionadas con la documentación y validación del proyecto.

Principalmente se utilizó para:

- comparar diferentes ideas de implementación del sistema de control
- generar una estructura inicial para un simulador tipo HMI que permitiera visualizar el comportamiento del sistema
- revisar y mejorar la redacción técnica de algunos apartados de la documentación en GitHub
- organizar la explicación de problemas encontrados durante la integración del sistema

**Descripción del objetivo:**

```
Utilizar la IA como herramienta de apoyo para comparar ideas de
implementación del sistema, generar un simulador sencillo que
representara el comportamiento del sistema tipo HMI y mejorar la
redacción técnica de la documentación del proyecto en GitHub.
```

## 4️⃣ Prompts relevantes utilizados

**Prompt 1:**

```
Ayúdame a crear un simulador sencillo tipo HMI para una cortina
automatizada controlada por un PLC Siemens LOGO donde se puedan
simular sensores de posición, el movimiento del motor y visualizar
los estados del sistema.
```

**Prompt 2:**

```
Corrige y mejora la redacción técnica de un texto para GitHub donde
se describe un problema en un sistema de control de una cortina
automatizada donde el motor se mueve demasiado rápido y sobrepasa
los sensores antes de detenerse.
```

---

## 5️⃣ Respuesta(s) obtenida(s) de la IA

La IA proporcionó:

- una estructura básica para un simulador que permitiera representar sensores y estados del sistema
- sugerencias para organizar la documentación técnica en GitHub
- mejoras en la redacción de algunas secciones del reporte
- ideas para explicar los problemas detectados durante la integración del sistema

Estas respuestas se utilizaron como base para discutir posibles soluciones dentro del equipo y mejorar la claridad de la documentación del proyecto.

## 6️⃣ Evaluación crítica del equipo (OBLIGATORIA)

### 6.1 ¿Qué parte fue útil?

La IA fue útil principalmente para:

- generar ideas iniciales para la estructura del simulador
- ayudar a organizar la documentación técnica del proyecto
- mejorar la claridad de la redacción en algunos apartados del repositorio de GitHub

También ayudó a identificar formas más claras de explicar los problemas técnicos encontrados durante las pruebas del sistema.

### 6.2 ¿Qué parte fue incorrecta, incompleta o no aplicable?

Algunas sugerencias de implementación del simulador eran demasiado complejas para el alcance del proyecto o no correspondían exactamente al funcionamiento del PLC utilizado.

Por esta razón fue necesario simplificar varias de las propuestas y adaptarlas a las condiciones reales del sistema.

## 8️⃣ Impacto del uso de IA en el proyecto

El uso de IA permitió ahorrar tiempo principalmente en la organización
de la documentación y en la generación de ideas iniciales para el
simulador del sistema.

También ayudó a mejorar la claridad de algunos apartados técnicos del
repositorio de GitHub. Sin embargo, las decisiones de ingeniería y la
validación final del sistema se realizaron mediante pruebas reales y
discusión entre los integrantes del equipo.

---

## 9️⃣ Aprendizaje del equipo sobre el uso de IA

El equipo aprendió que la inteligencia artificial puede ser una
herramienta útil para generar ideas iniciales y mejorar la
documentación, pero que en proyectos de ingeniería es necesario
verificar siempre la información y validar las soluciones mediante
pruebas reales.

También se aprendió que la IA funciona mejor como herramienta de apoyo
para organizar ideas y explicar conceptos, pero no debe sustituir el
análisis técnico ni la experimentación del equipo.
